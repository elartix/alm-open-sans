# open-sans Font Family

> open-sans Font Family Bower-ready (LESS-ready, and plain CSS-ready)

## How to use

0. Install it from Bower

   ```sh
   $ bower install [--save] 
   ```

1. Import it on LESS/CSS source code

    ```css
    @import "[bower_components]/open-sans(.css|.less)";
    ```
